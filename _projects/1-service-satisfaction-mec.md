This project studies how to allocate scarce wireless and edge-computing resources when a multi-cell MEC network cannot serve all users simultaneously.

## Research problem

How can a NOMA-based multi-cell MEC system jointly decide which users to serve and how to allocate subchannels, power, and computing resources while maximizing user service satisfaction?

## My role

Lead researcher and first author.

## Methods

- Joint user selection
- Subchannel scheduling
- Power allocation
- Computing resource allocation
- Convex optimization
- Evolutionary optimization

## Main results

The proposed method achieved a gap of less than 1% from exhaustive-search performance and improved average user service satisfaction by more than 50% compared with existing schemes.
