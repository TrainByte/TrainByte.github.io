## Overview

During the internship at China Mobile's Digital Intelligence Department, I worked on AI Agent memory mechanisms for long-horizon, multi-turn interaction scenarios. The work focused on how agents can preserve useful historical information while avoiding context forgetting, redundant memory accumulation, and inconsistent responses.

## Challenge

- Long conversations exceed context windows and make useful historical information harder to retrieve.
- Agent memory systems need to balance extraction quality, compression loss, forgetting, storage overhead, retrieval latency, and answer consistency.
- Memory management is no longer only a model-side issue when agents run across end, edge, and cloud resources.

## My Contributions

- Surveyed RAG-enhanced memory, hierarchical memory storage, self-evolving memory updates, memory extraction, compression, forgetting, consolidation, and dynamic updating.
- Reproduced and analyzed the Mem-alpha memory architecture.
- Abstracted memory selection, compression, and updating as a policy-optimization problem.
- Studied how reinforcement learning can support long-term memory maintenance.
- Connected agent memory management with end-edge-cloud task scheduling.
- Analyzed tradeoffs among memory storage, retrieval overhead, model inference cost, transmission delay, and response quality.

## Outcomes

- Built a structured technical map of long-horizon AI Agent memory pipelines.
- Established an experimental basis for reinforcement-learning-driven memory maintenance.
- Framed memory-aware LLM task scheduling as a practical systems problem across end, edge, and cloud resources.
