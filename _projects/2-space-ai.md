This project investigates distributed task routing in intelligent satellite computing networks, where topology, onboard resources, memory states, and link conditions vary over time.

- Model the coupling among task routing, memory resources, computing loads, and dynamic links.
- Use diffusion models to improve policy generation and high-dimensional state representation.
- Validate the method on a ground-based space-computing network test environment.
- Reduce end-to-end routing delay by more than 15% in experiments.
