This project studies how sensing accuracy, generation quality, communication capacity, and computing resources jointly affect user experience in ISAC-driven AIGC services.

- Establish a multi-dimensional sensing–computing–communication resource model.
- Design an LP-guided hierarchical reinforcement-learning framework.
- Use reinforcement learning for sensing and computing decisions and linear programming for communication allocation.
- Improve average service experience by more than 50% compared with schemes that focus only on generation quality.
