## Overview

Developed a distributed task-routing framework for intelligent space computing networks with dynamic topology, heterogeneous onboard computing resources, variable communication links, and distributed memory states.

## Challenge

- Space computing networks involve dynamic links, constrained onboard resources, and time-varying topology.
- Task-routing decisions depend on memory state, computing load, storage availability, and link conditions at the same time.
- High-dimensional dynamic states require routing methods that remain tractable under changing network conditions.

## My Contributions

- Modeled the coupling among task routing, node memory state, computing load, storage availability, and link conditions.
- Designed a distributed routing method for high-dimensional and dynamically changing network states.
- Introduced a diffusion-model-based policy representation to improve complex-state modeling and routing-decision generation.
- Evaluated the approach using a ground-based space-computing-network simulation environment.

## Technical Highlights

- Distributed routing under dynamic topology
- Memory-aware node-state modeling
- Diffusion-model-based policy representation
- Ground-based simulation evaluation

## Outcomes

The proposed approach reduced end-to-end routing delay by more than 15% in the evaluated simulation settings.
