## Overview

This project investigates memory-aware distributed task routing in intelligent space computing networks. In this setting, satellites and space nodes have different onboard computing capabilities, communication resources fluctuate over time, and each node may hold different memory or knowledge states relevant to AI tasks.

## Challenge

- Space network topology and link quality change dynamically.
- Routing decisions are coupled with node memory state, computing load, storage availability, and communication resources.
- High-dimensional dynamic states require policy generation methods that can capture complex dependencies.

## My Contributions

- Served as a main contributor for the memory-aware distributed routing mechanism.
- Built a joint model of task routing, memory resources, computing load, and link states.
- Designed a distributed routing and scheduling method for high-dimensional dynamic state spaces.
- Introduced a diffusion-model-based component to enhance strategy generation and complex-state representation.
- Completed experiments on a ground-based space computing network simulation system.

## Outcomes

The proposed method reduced end-to-end routing delay by more than 15% in the simulation environment. The related work has been submitted to a CAS Q2 journal.
