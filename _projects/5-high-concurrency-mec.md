This project addresses high-concurrency AI services in multi-cell MEC networks when communication and computing resources cannot serve all users simultaneously.

- Model user admission, bandwidth, subchannels, transmit power, computing resources, and service satisfaction.
- Jointly optimize user selection and communication-computing resource allocation.
- Improve average service satisfaction by more than 50% in resource-constrained scenarios.
