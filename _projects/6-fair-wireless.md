This project studies real-time wireless resource allocation that achieves long-term fairness without excessive loss of network efficiency.

- Model channel-estimation uncertainty and outage constraints.
- Transform long-term fairness objectives into real-time scheduling decisions.
- Design a reward-adaptive multi-agent reinforcement-learning framework.
- Improve long-term fairness by 13% while increasing network service rate by more than 10%.
