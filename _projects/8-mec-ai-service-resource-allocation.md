## Overview

This project studies high-concurrency AI services in multi-cell mobile edge computing networks. The work focuses on how to jointly select users and allocate wireless and computing resources when intelligent service requests arrive concurrently and resources are limited.

## Challenge

- Multi-cell MEC systems face concurrent service requests, wireless resource competition, and constrained edge computing capacity.
- User access decisions, bandwidth, subchannels, transmit power, computing allocation, and service satisfaction are tightly coupled.
- The scheduler needs to improve system service capability without sacrificing user experience.

## My Contributions

- Served as a main contributor and first author.
- Built a coupled model of user access, communication bandwidth, subchannel allocation, transmit power, edge computing resources, and service satisfaction.
- Designed a joint optimization algorithm for user selection, channel assignment, power control, and computing resource allocation.
- Completed simulation design, comparative evaluation, and paper writing.

## Outcomes

The proposed method improved average user satisfaction by more than 50% in the evaluated settings. The work was published in IEEE Transactions on Mobile Computing.
